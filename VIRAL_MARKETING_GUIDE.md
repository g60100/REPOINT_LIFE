### 스크립트 1: "돈 버는 앱"
```
[0-3초] 화면: 스마트폰 화면 클로즈업
"여러분 이거 아세요?"

[4-7초] 화면: 쿠팡 vs REPOINT 비교
"쿠팡에서 사면 0원 캐시백
REPOINT에서 사면 15% 캐시백!"

[8-12초] 화면: 10만원 → 15,000원 캐시백 애니메이션

```

"어제 커피 사고 1,500원 받음

[13-15초] 화면: 다운로드 링크

```

### 스크립트 3: "강남 핫플"
```

"강남 핫플 카페 왔어요"

## 📊 성과 측정 지표

### 트래킹 항목
1. **조회수** (Views)
2. **클릭수** (Clicks)
3. **전환율** (Conversion Rate)
4. **가입자수** (Signups)
5. **ROI** (Return on Investment)

### 목표 KPI
```
틱톡/릴스:
- 조회수: 100만+
- 클릭율: 5%
- 가입 전환: 10%
→ 5,000명 가입

유튜브 쇼츠:
- 조회수: 200만+
- 클릭율: 3%
- 가입 전환: 5%
→ 3,000명 가입

인스타그램:
- 도달: 50만+
- 클릭율: 2%
- 가입 전환: 3%
→ 300명 가입

총 예상: 8,300명 가입
```
