## 💰 수익 계산 예시

### 예시 1: 팔로워 10만 유튜버
```
월 활동:
- 영상 4개 제작 (20만원 × 4 = 80만원)
- 추천 링크로 200명 가입
- 그 중 100명이 월 10만원 구매

수익:
- 콘텐츠 제작비: 80만원
- 추천 수수료: 100명 × 10만원 × 10% = 100만원
- 성과 보너스: 50만원
총 수익: 230만원/월
```

### 예시 2: 팔로워 5만 틱톡커
```
월 활동:
- 영상 10개 제작 (10만원 × 10 = 100만원)
- 추천 링크로 500명 가입
- 그 중 200명이 월 5만원 구매

수익:
- 콘텐츠 제작비: 100만원
- 추천 수수료: 200명 × 5만원 × 8% = 80만원
- 성과 보너스: 200만원
총 수익: 380만원/월
```

## 🚀 즉시 시작 가이드

### 1단계: 등록 (5분)
```
POST /api/influencer/register
{
  "influencer_name": "홍길동",
  "platform": "youtube",
  "channel_url": "youtube.com/@example",
  "follower_count": 50000
}
```

### 2단계: 추천 링크 받기
```
응답:
{
  "referral_code": "INF12345ABC",
  "referral_url": "repoint.life/signup?inf=INF12345ABC",
  "commission_rate": 0.08
}
```

### 3단계: 콘텐츠 제작
```
- 제공된 스크립트 활용
- 추천 링크 포함
- 업로드 후 URL 등록
```

### 4단계: 수익 확인
```
GET /api/influencer/dashboard
- 실시간 통계 확인
- 수익 현황 확인
```

### 5단계: 정산 요청
```
POST /api/influencer/settlement/request
- 월말 정산 요청
- 3일 내 지급
```
