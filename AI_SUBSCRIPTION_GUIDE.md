## 📱 사용 방법

### 1. 구독 신청

```typescript
POST /api/subscription/subscribe
{
  "plan_id": 1,
  "payment_method": "points"
}
```

**응답:**
```json
{
  "success": true,
  "subscription_id": 123,
  "credits": 35
}
```

---

### 2. AI 콘텐츠 생성

```typescript
POST /api/subscription/ai/generate
{
  "content_type": "video_ad",
  "prompt": "강남 카페 신메뉴 아메리카노 광고 영상, 모던하고 세련된 분위기",
  "ai_model": "runway"
}
```

**응답:**
```json
{
  "success": true,
  "content_id": 456,
  "url": "https://cdn.repoint.life/ai/456.mp4",
  "remaining_credits": 34
}
```

---

### 3. SNS 자동 발송

#### 즉시 발송
```typescript
POST /api/subscription/sns/post-now
{
  "content_id": 456,
  "platform": "instagram",
  "post_text": "신메뉴 출시! 🎉",
  "hashtags": "#강남카페 #신메뉴 #아메리카노"
}
```

#### 예약 발송
```typescript
POST /api/subscription/sns/schedule
{
  "content_id": 456,
  "platform": "instagram",
  "post_text": "신메뉴 출시! 🎉",
  "hashtags": "#강남카페 #신메뉴 #아메리카노",
  "scheduled_at": "2026-02-18 10:00:00"
}
```

## 📊 대시보드 기능

### 내 구독 현황
```
GET /api/subscription/my
```

**확인 가능:**
- 현재 플랜
- 남은 크레딧
- 구독 기간
- 자동 갱신 여부

---

### 생성한 콘텐츠
```
GET /api/subscription/ai/my-content
```

**확인 가능:**
- 생성한 모든 콘텐츠
- 콘텐츠 타입
- 생성 날짜
- 다운로드 링크

---

### SNS 발송 내역
```
GET /api/subscription/sns/my-posts
```

**확인 가능:**
- 발송한 모든 포스팅
- 플랫폼별 분류
- 발송 날짜
- 포스팅 URL
